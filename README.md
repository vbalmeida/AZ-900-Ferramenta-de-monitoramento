# AZ-900-Ferramenta-de-monitoramento

# Ferramentas de Monitoramento do Azure


# Assistente do Azure

Analisa recursos implantados do Azure e faz recomendações com base nas práticas recomendadas para otimizar as implantações do Azure. Podendo nos recomendar determinadas ações que devem ser tomadas em recursos.

# Integridade do Serviço do Azure

Serve para nos informar o status geral do Azure, onde posso verificar se algum serviço está indisponível ou quando isso vai acontecer.

Status do Azure: visão global da integridade de todos os serviços do Azure em todas as regiões do Azure.
Se um serviço estiver enfrentando um problema em uma região que você não está usando, ele não aparecerá para você no status.

# Resource Health: Ele fornece informações  sobre a integridade de seus recursos de nuvem individuais.

 # Azure Monitor 

É uma plataforma de monitoramento completa fornecida pela Microsoft para ambientes em nuvem e híbridos. Ele permite que os usuários coletem e analisem dados de monitoramento de aplicativos, infraestrutura e rede em tempo real, oferecendo insights valiosos para garantir o desempenho, a disponibilidade e a segurança dos serviços implantados no Azure, em ambientes locais e em outras nuvens públicas.
